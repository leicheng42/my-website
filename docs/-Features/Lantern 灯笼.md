# 灯笼 Lantern

在您的地块中会发光的功能。灯光的颜色是自定义的。
此功能是在版本v4.6.0中添加的。
**要让灯笼工作，您必须在地块上激活 [light-mapping](https://wiki.cryptovoxels.com/Parcels/light-map)**

> 灯笼仍处于测试阶段。
{.is-info}
