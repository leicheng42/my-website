# 绑定快捷键

在这个页面中，我们介绍Cryptovoxels的绑定快捷键！

## 按键控制

|按键 | 功能 |
|-----|-----|
| `H` | 切换帮助窗口 |
| `WASD` | 移动键（前进、左移、后退、右移）|
| `Shift` | 跑步|
| `Space` | 跳跃 |
| `Enter` | 打开聊天菜单|
| `F` | 切换飞行模式|
| `V` | 按住进行语音聊天|
| `I` | 切换检查模式|
| `G` | 切换表情菜单|
| `C` | 切换第三人称摄影机视角|
| `Z` | 切换缩放|

## 建筑

|按键 | 功能 |
|-----|-----|
| `TAB` | 切换建筑菜单 |
| `1-9` | 选择方块瓦片|
| `点击` | 放置方块|
| `点击 + 拖动` | 放置多个方块 |
| `SHIFT + 点击` | 删除方块 |
| `CTRL + 点击` | 绘制方块颜色 |
| `SHIFT + 点击 + 拖动` | 删除多个方块|
| `R` | 复制功能|
| `E` | 编辑功能 |
| `M` | 移动功能|
